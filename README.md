🌦 Weather API (Python)

A simple Weather API built using Python and Flask that allows users to retrieve weather information for a given location.

This project demonstrates how to:

Build a REST API using Flask

Connect to an external weather service

Handle JSON requests and responses

Structure a simple backend application

🚀 Features

Get current weather by city name

Returns data in JSON format

Simple and beginner-friendly structure

Easy to extend with more features (forecast, humidity, wind speed, etc.)

🛠 Technologies Used

Python 3

Flask

Requests library

External Weather API (e.g., OpenWeather)
